Sketchy Draw

Sketchy Draw is a simple powerful vector design application. All of the tools you need to create illustrations to beautiful works of art.

Which formats are supported ?

Sketchy Draw natively supports PDF and many others
like SVG, SVGZ, EPS, EPSI ...
